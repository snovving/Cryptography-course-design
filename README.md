RSA、ELGamal 等签名的实现都是依靠了 GMP 库，我个人的实现其实就如 ex1 的 Miller-Rabin 算法，用的是系统时间生成随机数种子，都不是理想中的那么“大”，而 GMP 库是有专门的函数来生成大素数的（梅森旋转算法），有时间会专门写一篇博客来分析，代码很简单，就不放了。

最后就是信息的完整性验证，这个我是用 py3 socket 编程实现的（不到 15 min 赶工完成，勿喷，还是未完成版，仅供参考），信息完整性验证这个过程简单描述就是——

发送方和接收方共享一个秘密密钥，发送方发送原始数据和 HMAC 过的密文，接收方接收两者，用自己本地的密钥对原始数据 HMAC，然后对比两个 HMAC 是否相等来判断信息是否遭到篡改，所以个人就突发奇想用了计网的 socket 来实现，因为剩余课设的时间很少了，本来还想添加一个钩子来截取 socket 模拟黑客改下数据校验，这部分就留到有空的时候再做吧 XD
